# SEO & Performance Skill

## Guidelines
1. **Dynamic Meta Descriptions**: Each route must have tailored meta descriptions using keywords like "Terapia Transpersonal" and "Meditación".
2. **Open Graph**: Always use `og:image` and `og:title` on all public pages for social sharing.
3. **Semantic HTML**: Strict usage of H1-H6 hierarchy. Always ensure there is exactly one `<h1>` per page for clarity to search engines.
4. **Accessibility**: Implement ARIA tags appropriately for interactive elements like modals or hidden panels.
5. **Asset Optimization**: Ensure fast loading properties and caching headers. Provide `lazy` loading for elements below the fold.
